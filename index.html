Инструкция для арт-директора. Как нарисовать и подготовить рамки для всех UI-элементов игры.

✨ Как работает рамка

Каждая рамка — это 3 файла PNG. Верх и низ фиксированы, середина растягивается под контент.

👑
1_top.png — «корона» Верхний орнамент/декор. Не тянется.
фиксированная
🔁
2_middle.png — «обои» Повторяется вниз сколько нужно. Должна быть бесшовной!
повторяется
🏛️
3_bottom.png — «подвал» Нижний декор. Не тянется.
фиксированная
Рисуй всё в @2x (Retina). В config.txt пиши вдвое меньше — PNG 120px → topHeight = 60

🔧 Нарезка в Фотошопе

1
Создай холст Ширина 800px, высота на твой выбор (например 400px). Фон — прозрачный.
2
Нарисуй рамку Орнамент сверху и снизу. Середина — простая, без узоров (только фон + бордюры по бокам).
3
Поставь направляющие Просмотр → Новая направляющая
Пример (холст 400px, декор 120px):
• Горизонтальная на 120px
• Горизонтальная на 280px (400−120)
4
Нарежь по направляющим Инструмент Нарезка (C) → Просмотр → Нарезать по направляющим
5
Экспортируй Файл → Экспортировать → Сохранить для Web
Формат: PNG-24, галочка Прозрачность
6
Переименуй файлы Верхний → 1_top.png
Средний → 2_middle.png
Нижний → 3_bottom.png

🧵 Как проверить бесшовность 2_middle.png

→
Открой 2_middle.png в Фотошопе Фильтр → Другие → Сдвиг → Вертикаль = высота/2, Переносить
Видишь шов? Замажь Clone Stamp. Шва нет — готово ✨
Если середина однотонная — шва нет автоматически. Проблемы только с узорами.

✨ Украшения (акценты)

Декоративные PNG поверх рамки — корона, завитки, кристаллы. 6 позиций:

top-leftверх-лево
top-centerверх-центр
top-rightверх-право
bottom-leftниз-лево
bottom-centerниз-центр
bottom-rightниз-право
1
Нарисуй PNG с прозрачным фономУгловые: 40–80px. Центральные: 60–120px (@2x)
2
Назови по позицииtop-center.png, bottom-left.png и т.д.
3
Положи в папку accents/И пропиши в config.txt:
accent.top-center.width = 56px
accent.top-center.height = 32px
accent.top-center.offsetY = -2px
Без параметров в config.txt акценты не отображаются — движок не сканирует папку сам.

⚠️ Частые ошибки

Положила PNG, но не прописала в config.txt Движок не знает что рамка есть — покажет стандартный фон.
Разная ширина у трёх файлов Рамка ломается. Все три должны быть одной ширины.
Белый фон вместо прозрачного Будет белый прямоугольник в игре. PNG-24 + галочка Прозрачность.
Не проверила бесшовность 2_middle.png Виден шов когда текст длинный. Проверяй через Фильтр → Сдвиг.
Забыла делить на 2 для Retina PNG 120px → topHeight = 60, не 120.
Сделала speaker-badge такой же как dialogue Плашка с именем будет огромной. Это маленькая тонкая рамочка.
